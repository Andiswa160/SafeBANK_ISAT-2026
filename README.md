# SafeBANK 💰
   
   A Power Apps banking application with live SharePoint integration
   
   **🔗 Live Demo:** [https://apps.powerapps.com/play/e/default-543f2205-fcdc-4c35-ad7a-ea6bd74ca885/a/c00cbe28-372e-4c90-956b-f60ac03294a6?tenantId=543f2205-fcdc-4c35-ad7a-ea6bd74ca885&hint=e967521b-cdf1-4a29-a980-27f1bee4e1f9&sourcetime=1777074193409]
   
   **✨ Features:**
   - Real-time balance from SharePoint Lists
   - Deposit/Withdraw functionality with Patch() 
   - Input validation + error handling
   - Clean, mobile-friendly UI
   
   **🛠️ Built With:**
   - Microsoft Power Apps
   - SharePoint Lists
   - Power Fx
